# TaskMaster-Pro

A simple, responsive, and feature-rich to-do list application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Edit tasks
- Delete tasks
- Filter tasks (All, Pending, Completed)
- Export tasks to text file
- Statistics display
- Local storage support
- Responsive design
- User notifications

## How to Run

1. Download all three files: `index.html`, `style.css`, and `script.js`
2. Place them in the same directory
3. Open `index.html` in any modern web browser

## Usage Instructions

1. **Add a Task**: Type your task in the input field and press "Add Task" or Enter key
2. **Complete a Task**: Click the checkbox next to any task
3. **Edit a Task**: Click the edit (pencil) button, modify the text, and save
4. **Delete a Task**: Click the delete (trash) button and confirm
5. **Filter Tasks**: Use the filter buttons to view All, Pending, or Completed tasks
6. **Export Tasks**: Click Export button to download tasks as text file

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Media Queries)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- CSS Animations
